# MILO

Modular Intelligence for Life Organization.

MILO is a tablet-first, ADHD-friendly personal dashboard built with React,
TypeScript, Vite, Redux Toolkit, RTK Query, and React Router.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run preview
```
